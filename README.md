# Sportea App

A React Native Expo application for sports enthusiasts.

## Project Structure

```
assets/
src/
  components/
  screens/
  navigation/
  hooks/
  utils/
```

## Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/ramos333oz/Sportea-App.git
```

2. Install dependencies:
```bash
npm install
```

3. Run the application:
```bash
npx expo start
```
